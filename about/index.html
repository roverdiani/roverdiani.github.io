ABOUT

Hey there!

My name is Rodrigo Verdiani (aka Roverdi), and this is my blog :D

I’m a twentyish (okay, almost thirty xD) years old Brazilian, with a passion for programming. I’m currently working as a frontend developer (React).

My main tech interests are (in no particular order):

  • Graphics Programming
  • Game Engine Programming
  • AI
  • Quantum Computing
  • Retro tech

Besides that, some of my hobbies are:

  • Role Playing Games
  • Electronics (mainly audio stuff and old digital tech, like Z80s, 6502, etc)
  • Music
  • Photography

The main idea behind this website is to share things that I learn, love and(or) may be useful for others. It also can be useful as a portfolio, or a journal of the things I’ve tinkered with.

On a previous version of this blog, I wrote that I couldn’t decide which was my favorite programming language. Well, nowadays, I tend to write most of my personal code in C. I find that I can focus more on the logic and structure side of things, given the relative simplicity of the language.

I hope that you find my content interesting, and if you have any suggestions, found something wrong or have any problem, please let me know. As English is not my main language, I would appreciate some feedback. :P

Thank you reading all of this! See ya!
